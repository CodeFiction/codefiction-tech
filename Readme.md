# Codefiction Homepage

[Codefiction Home Page](http://www.codefiction.tech) homepage source code.

## Requirements
 - npm v4+
 - ng v1.2+

## Build and Run

```sh
npm install
ng serve (local dev)
```

Navigate to `http://localhost:4200/`
